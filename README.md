# AutoCAD Manage

## Clone

```bash
git clone https://github.com/565353780/autocad-manage.git
cd autocad-manage
```

## Install

```bash
./setup.sh
```

for Windows

```bash
./setup_win.sh
```

## Local Mode

start AutoCAD application

### trans DWG to DXF

```bash
python demo_win.py
```

### DXF layout detect

```bash
python demo.py
```

## Server Mode

### Run Server

start AutoCAD application

run autocad server on Windows

```bash
python win_server.py
```

run flask server

```bash
python flask_server.py
```

### Run Client

```bash
python client.py
```
