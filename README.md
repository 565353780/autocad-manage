# AutoCAD Manage

## Clone

```bash
git clone https://github.com/565353780/autocad-manage.git
cd autocad-manage
```

## Install

```bash
./setup.sh
```

for Windows

```bash
./setup_win.sh
```

## Run Server

start AutoCAD application

run autocad server on Windows

```bash
python win_server.py
```

run flask server

```bash
python flask_server.py
```

## Run Client

```bash
python client.py
```

## Usage

look at

```bash
demo.py
demo_win.py
```
